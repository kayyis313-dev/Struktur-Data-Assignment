## Guided 

### 1. [Nama Topik]

```C++
#include <iostream>
using namespace std;

int main () {
    int n;
    const float pi = 3.14;

    cout << "Masukan Angka: ";
    cin >> n ;

    cout << "Angka dikeluarkan: " << n << endl;
    cout << "Nilai konstanta pi: " << pi << endl;
    return 0;
}

```
